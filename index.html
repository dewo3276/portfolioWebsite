She Kills Monsters

Dragon Head

A mechanical design overview of a dragon skull creation for University Theater and Dance

This project was designed by myself and coworker Hannah Male for the play "She Kills Monsters". It was designed with the intention of being on a green screen and edited for film during the COVID-19 pandemic. The whole head was painted white to allow for the lighting crew to add different colors to it during filming to produce the five headed hydra so five different heads didn't need to be created. The mechanical requirements for the show was for it to have pitch in one direction and yaw at the base of the neck and connection point behind the green screen via a custom built mounting device. The skull also had to have a movable jaw for line delivery which was operated by a pulley system and lizard frills also operated by a pulley system. 

Broadway Christmas Carol

Scenic elements created for Broadway Christmas Carol

These hide away stairs provided an ease of access for actors in the production of Broadway Christmas Carol to dance on the tables for multiple musical numbers. These tables also functioned as many different set pieces as they were designed to be mobile with a functioning wheel lift system. Trim was added after the initial construction was done and tests were conducted to ensure safety as well as functionality by the actors. To better conceal the ladder, a hinged door held by magnets was added. After some failures with this system, these magnets were replaced as the tap dances that occurred on that table cause the doors to swing open at random intervals. There was also a problem with the trim interfering with the rotation of the wheels, so cuts were made to allow for full spin of the wheels.

Hair

Scenic elements created for Hair

This set was created for the musical hair. It was created by reusing some pieces from previous shows and adjusting some of these pieces. A rectangular frame was cut from a previous platform, some dimensions were changed throught plasma cutting and MIG welding new pieces and faced with 3/4" plywood cut with a jigsaw. Turf and specially built stair units were stagged to give the hill coming up to the bridge a safe way for actors to ascend as well as give the illusion of a hill. The handrail was created using a tube bender. The benches and other set pieces were modified to be able to easily be moved by actors during scenes. The benches were the plastic benches you would see at parks that are quite heavy and neede to be replaced with some 2x4's.

Hecuba

Scenic elements created for hecuba

Hecuba was created using many different rigging techniques along with some foam carving and painting. The beams across the proscenium were built with long sonotube with foam strips cut in long rectangular strips glued around the diameter. It was then covered in muslin fabric after the foam was carved and given some texture to appear like misshapen logs. These were then hung from the front linesets using genie lifts and rope wrapped around the pieces. The backdrops served as a way to create many different levels within the scenes when the dream sequences with Hecuba occur as her dead children haunt her dreams. These were backlit to create haunting silhouettes.

Great Expecations

Scenic elements created for Great Expecations

This set was created for the black box theater in the loft space. The space was moved for a 3/4 round and crafted from some stock platforms as well as custom made platforms. The flooring took the longest of everything because it was created using a brick pattern from tape. Windows were hung from the grid but cut using a router to give a rounded edge to them. Phi Delta Theta Letters

Steel lettering created and installed

These letters were created for the Phi Delta Theta Colorado Alpha chapter and were installed as of 9/5/2020. This was a commissioned project from the Alumni Bill Mark. As one of the last re-founding fathers still in the Colorado Alpha chapter, this project meant a lot to me and my brothers. The letters themselves were cut from 16 gauge steel sheets using a plasma cutter. To achieve precise shapes, I cut out stencils I designed in Rhino on the laser cutter using plywood. Once the stencils were cut, they were placed on the sheet metal and clamped securely in place so I could use them as a guide for the plasma cutting tool. Anchors were added to the back of the letters, spray painted, placed into pre-drilled holes in the house, and secured with construction adhesive.

Self Destructing Clock

Capstone project

This project was created as my Capstone within the CTD program. The clock was a self-destructing art piece meant to give a qualitative aspect to time via quantitative means.
Through a google chrome extension that tracks zoom time and a wifi connected door monitoring system, the clock tracked when people left the virtual world for the real one and vice versa. The clock had two main functions: to spray paint itself any time the door tracker was pushed and to hit itself in 30 minute increments based on how long users stayed on zoom.

Cannon Fodder

A physical computing game

Cannon Fodder was created as a midterm project to demonstrate mastery over circuitry and coding within the arduino IDE. This project consisted of 2 potentiometers, 2 servo motors, 2 dc motors, a solenoid, neopixel strips, and 4 limit swithces. The cannon itself was a movable piece that was able to tilt up and down and rotat side to side as well as fire a ping pong ball from it. The goal of the game was to land as many ping pong balls into the score board and get the most points. The scoreboard had a few functions that kept track of score as well as let the user know what their current score was.
